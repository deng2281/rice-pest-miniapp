# 水稻病虫害识别与预警微信小程序系统策划

## 1. 项目概述
本项目旨在开发一款简易的水稻病虫害识别与预警微信小程序，利用阿里大模型（Qwen系列）的强大视觉识别能力，帮助农户快速诊断水稻病虫害问题，并提供相应的防治建议。

## 2. 核心功能模块

### 2.1 图片上传模块
- **功能描述**：允许用户从手机相册选择图片或直接拍摄水稻病害部位的照片。
- **技术实现**：
  - 使用微信小程序 `wx.chooseMedia` 接口获取图片。
  - 图片需进行压缩处理（限制大小，如 2MB 以内），转为 Base64 格式以便传输给大模型接口。

### 2.2 智能识别模块
- **功能描述**：调用阿里云百炼大模型接口，对上传的图片进行分析。
- **模型配置**：
  - **模型名称**：`qwen-vl-max` 或用户指定的 `qwen3.5-flash`（注：通常视觉任务推荐使用 VL 系列模型，如 `qwen-vl-max`，若 `qwen3.5-flash` 支持视觉输入则直接使用）。
  - **Prompt 设计**：设定系统角色为“农业专家”，要求输出格式包含：病虫害名称、置信度、症状描述、防治建议。
- **接口鉴权**：使用提供的 API Key 进行鉴权。

### 2.3 结果展示模块
- **功能描述**：将模型返回的分析结果结构化展示给用户。
- **UI 设计**：
  - 顶部显示上传的原图。
  - 下方卡片式展示识别结果（病名高亮）。
  - 详细的防治措施列表。

### 2.4 预警建议模块（简易版）
- **功能描述**：根据识别结果，给出紧急程度预警（如：需立即处理、持续观察）。

## 3. 技术架构

- **前端**：原生微信小程序 (WXML, WXSS, JS)
- **后端/模型层**：阿里云 DashScope API (兼容 OpenAI 接口格式)
- **数据流**：小程序 -> 阿里云 API -> 小程序

## 4. 开发计划

1.  **环境准备**：配置微信开发者工具，设置“不校验合法域名”（用于开发调试）。
2.  **界面开发**：编写 `pages/index` 的上传界面和结果展示界面。
3.  **逻辑实现**：
    - 封装 `uploadImage` 函数。
    - 封装 `callAIModel` 函数（处理 Base64 编码和 HTTP 请求）。
4.  **联调测试**：使用真实水稻病害图片进行测试。

## 5. API 接口定义 (拟定)

- **URL**: `https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`
- **Method**: `POST`
- **Headers**:
  - `Authorization`: `Bearer <YOUR_API_KEY>`
  - `Content-Type`: `application/json`
- **Body**:
  ```json
  {
    "model": "qwen-vl-max", 
    "messages": [
      {
        "role": "system",
        "content": "你是一个专业的水稻病虫害专家。请分析图片中的水稻病虫害情况，给出病害名称、症状分析和防治建议。"
      },
      {
        "role": "user",
        "content": [
          {"type": "text", "text": "请分析这张水稻图片。"},
          {"type": "image_url", "image_url": {"url": "data:image/jpeg;base64,..."}}
        ]
      }
    ]
  }
  ```
